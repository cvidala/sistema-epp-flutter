## What Was Built

### .github/workflows/test.yml

A 7-step GitHub Actions workflow that runs on every push and pull_request targeting main:

1. `actions/checkout@v4`
2. `subosito/flutter-action@v2` with Flutter 3.24.x stable, pub cache enabled
3. `flutter pub get`
4. `flutter analyze --no-fatal-infos` (Analyze step — fails job on lint errors)
5. `flutter test test/unit/ test/widget_test.dart --coverage` (54 unit+widget tests)
6. `flutter test test/integration/supabase/ test/integration/e2e/ --tags integration` (17 tests, 11 secrets injected)
7. `actions/upload-artifact@v4` with `if: always()` — uploads coverage/ directory as "coverage-report"

Hard constraints enforced: `timeout-minutes: 15`, no `continue-on-error` anywhere.

### .gitignore additions

Added under `# Coverage reports` comment:
- `lcov.info` — root-level file generated by `flutter test --coverage`
- `coverage/html/` — directory from manual `genhtml` invocation

Pre-existing exclusions confirmed: `/coverage/`, `.env.test`.
